打開你的 Notion 資料庫頁面，URL 格式為 notion.so/<workspace>/[Database ID]?v=…， 複製 32 位的 ID（不含 ?v= 後面的部分）。

使用說明

  1. 自動擷取：打開 claude.ai 並開始對話，擴充功能會自動擷取並儲存。
  2. 匯出 Markdown：點擊 Popup 中的「⬇ MD」按鈕，即可下載完整對話的 .md 檔案。
  3. 同步 Notion：
    1. 在 notion.so/my-integrations 建立 Integration
    2. 建立一個有 Title 欄位的 Notion 資料庫
    3. 在資料庫頁面右上角「…」→「連結整合」→ 選擇你的 Integration
    4. 將 Token 和 Database ID 填入上方設定並儲存
    5. 回到 Popup，點擊「Notion」即可同步
